CSS

css is the best part of creating web-pages this is when you can decide how you want your webpage to look like visually. For instance alot of parameters that are placed in HTML like the height tag in the img placeholder above can be specified in CSS. So whats the advantage of doing it at the top. The advantage is its all in one spot, and there are a ton more things you can change in css that isnt even possible with just HTML alone. So lets get started:

A great practice to get into is placing content in DIV tags as you can change pretty much whatever you want within them. Then to use CSS you first need a name for DIV tag such as divName. In the HTML body you would declare this just like this. <div class="divName"> Now up in the <style> section of your webpage, which should be up in the <head> section you can type .divName{} and anything that you place in the brackets will appear as if it were incorporated right into the webpage. .divName{width: 50%;} would give the DIV tag with the class-name divName a width attribute of 50%.
